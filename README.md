# dotfiles

My dotfiles managed by chezmoi

## first time set up

### Bash
```sh
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/ruhigwolf/dotfiles/HEAD/install.sh)"
```

### Powershell
```ps1
. { iwr -useb https://raw.githubusercontent.com/ruhigwolf/dotfiles/HEAD/install.ps1 } | iex
```
